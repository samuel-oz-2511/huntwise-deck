Sentriq runs before the application enters the ATS. The trust signal travels with it through the pipeline. No blocking. No friction.

Layer 1: Passive

Behavioral metadata collected as the applicant fills the form. Invisible. No friction. Keystroke dynamics, mouse movement, fill patterns, timing anomalies.

Catches: mass-apply bots, CV overfitting, skills anomaly, timeline fraud
Layer 2: Active Challenge

Triggered only when Layer 1 flags an anomaly. A short task framed as part of the application: behavioral fingerprint scored against content quality. A proxy cannot replicate both.

Catches: proxy impersonation, AI-assisted response, profile distortion
Layer 3: Human Review

Sentriq analyst reviews the full signal stack. The employer makes the final call. Sentriq never rejects unilaterally. Full audit trail for regulatory defense.

Zero automated rejections. Human-in-the-loop is architecture, not a disclaimer.

Platform-agnostic. No ATS modification required.
Output is a trust signal per application, not a pass/fail decision.

The Engagement Scope

What PwC would be scoping. And why it is new territory.

No existing SOC 2 framework covers behavioral application trust at a neutral pre-ATS layer. PwC would be writing the control environment for a new category.

Scope Area 1: Behavioral Data Collection

Keystroke dynamics, mouse movement, form-fill timing. Processing methodology, retention schedule, access controls, deletion workflows. Data subject rights framework (GDPR + EU AI Act overlay).

Scope Area 2: Cross-Company Knowledge Graph

Pseudonymized cross-employer signal aggregation. Neutral-party data architecture (no employer data sharing). Consent framework, data minimization controls, bias mitigation documentation.

Scope Area 3: Challenge Layer

Async task delivery pipeline. Scoring methodology and calibration. Human reviewer access controls. Non-discrimination controls and audit trail per application decision.

Scope Area 4: Trust Signal Output

Signal generation logic, ATS transmission, storage, and usage controls. End-to-end traceability from raw behavioral data to employer-facing signal. Immutable log for regulatory defense.
