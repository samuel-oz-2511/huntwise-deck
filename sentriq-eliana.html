How It Works

Signal at submission. Continuous through every stage after.

Sentriq enriches every inbound application with behavioral, consistency, and gap signals while the recruiter stays in their ATS. No candidate is automatically rejected at any stage.

Layer 1

Behavioral Baseline

Captured passively while the application is filled: timing, edit patterns, tool signatures. Zero friction, no changes to the form.

Layer 2

Async Clarification

Triggered only when Layer 1 finds a gap worth surfacing: a short, specific question tied to what was detected. Candidates with strong applications never see it.

Layer 3

Human Review

When Layer 2 flags divergence, a Sentriq analyst reviews the full signal stack and surfaces a trust signal alongside the application. The recruiter decides.

Gap Assessor
The same signal engine re-runs after every hiring stage: HR screen notes, test scores, interview transcripts, manager assessments. Each new data point is evaluated against what the candidate originally claimed, so gaps surface before offer, not after hire.
