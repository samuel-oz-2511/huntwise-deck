02 / The Mark

Pending
Candidate submitted. Sentriq has not yet processed the signals.
Scanning
Signal arcs pulse. Sentriq is processing behavioral and identity layers.
Cleared
The bracket resolves to the V. The same form. The arms were always the V.
The V is latent in the bracket. The two bracket arms are the two arms of the V. When a candidate clears, the mark does not switch to a new symbol. The same form resolves. There is no product reveal, no celebration icon. The bracket was always a gate. The V is what you see when the gate is passed.
Three contexts. Three forms. The primary static mark is brackets only: pure form, no arcs, no dot. Use everywhere the brand appears: decks, nav, email headers, social. The scanning animation (arcs pulsing above the brackets) is reserved for product UI only, the Scanning state in the candidate review interface. The V badge (amber square with white V) is the standalone trust seal: ATS integrations, email footers, cleared confirmation screens.

10 / Deck Pre-Ship Checklist

Run every command. Count must pass. Do not push until all pass.
1
No em dashes
grep -c "—\|&mdash;" file.html
Must return 0. Any count above 0 is a hard stop.
Must = 0
2
Correct logo SVG in every instance
grep -c 'rx="2" fill="#E8A020"' file.html
Must return 4 per logo instance. Gate + page header = 8 minimum. If below: SVG is missing or wrong somewhere.
Must = 8+
3
No invented SVG shapes in logo area
grep -n "logo" file.html | grep -i "circle\|path\|polygon"
Must return nothing. Logo is four rects only. Circles or paths in logo context = wrong SVG.
Must = 0
4
No sentriq.io domain
grep -c "sentriq\.io" file.html
Must return 0. Domain is always sentriq.co.
Must = 0
5
No emojis in CISO or investor decks
grep -c "&#&#1[0-9]\{5\};\|&#9[0-9]\{3\};" file.html
Must return 0 for CISO and investor decks. Emojis break the security/institutional register. HR decks may use sparingly.
Must = 0
6
Gate title does not name the recipient
Read the gate h2. If it contains a company name, bank name, or person name: change it to "Restricted Security Briefing" or "Confidential Briefing." The recipient discovers personalization inside the deck, not on the door.
Manual
7
Amber hierarchy: one dominant per section
Scan each section visually. If more than one element is amber and full-brightness, one of them must recede to white or rgba(255,255,255,0.85). Amber means "look here." If everything is amber, nothing is.
Visual
8
All dollar figures validated against financial model
Every $ figure must be sourced from sentriq-financial-model.html before use. Key numbers: Tier 1 ACV $240K, Tier 2 $84K, blended $150K, design partner year 1 discount 20% (Tier 1 $192K, Tier 2 $67K). No invented figures.
Manual
