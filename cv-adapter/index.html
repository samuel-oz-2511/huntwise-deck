1
Accept your GitHub invitation
One time
▼

The code lives in a private repository on GitHub. You need a GitHub account and an invitation before you can download it.

If you do not have a GitHub account yet:

  1. Go to github.com/signup and create a free account.
  2. Verify your email address.
  3. Send your GitHub username back to whoever shared this link with you so they can invite you.

Once you have been invited:

  1. Check your email for a message from GitHub with the subject "You have been invited to collaborate on samuel-oz-2511/cv-adapt-engine".
  2. Click View invitation in that email.
  3. On the GitHub page that opens, click Accept invitation.
You will see the repo at github.com/samuel-oz-2511/cv-adapt-engine once your invite is accepted.
2
Install Git and Python
One time
▼

You need two things: Git (to download the repo) and Python 3.10+ (to run the app).

Check if you already have them:

Terminal — open with Cmd+Space, type Terminal, press Enter 
git --version && python3 --version

If both print version numbers (e.g. git version 2.x and Python 3.11.x), you are ready. Skip to step 3.

Install Git (if missing):

Terminal 
xcode-select --install

A dialog will appear — click Install. This takes a few minutes.

Or download Git directly: git-scm.com/downloads/mac

Install Python 3.10+ (if missing):
Download the macOS installer from python.org/downloads and run it.

Homebrew users: brew install python3

Open Command Prompt: press Win+R, type cmd, press Enter.

Command Prompt 
git --version && python --version

If both print version numbers, skip to step 3.

Install Git: Download from git-scm.com/downloads/win. Run the installer, keep all defaults.

Install Python: Download from python.org/downloads.

Critical: On the first screen of the Python installer, check "Add Python to PATH". The app will not run without this.

After installing both, close Command Prompt and open a new one, then re-run the version check above.

3
Download the repository
One time
▼

Clone (download) the repo to your Desktop. Open a terminal or command prompt and run:

Terminal 
cd ~/Desktop
git clone https://github.com/samuel-oz-2511/cv-adapt-engine.git
cd cv-adapt-engine
Command Prompt 
cd %USERPROFILE%\Desktop
git clone https://github.com/samuel-oz-2511/cv-adapt-engine.git
cd cv-adapt-engine

Git will ask for your GitHub username and password. For the password, use a Personal Access Token — not your GitHub account password. Here is how to get one in 60 seconds:

  1. Go to github.com/settings/tokens/new
  2. Give it a name like "cv-adapter"
  3. Set expiration to 90 days or No expiration
  4. Check the repo checkbox (first option under "Select scopes")
  5. Click Generate token at the bottom
  6. Copy the token — paste it as your password when Git asks
GitHub no longer accepts your account password for git clone. The token is required.

After cloning, confirm you see a folder called cv-adapt-engine on your Desktop with app.py and install.sh (or install.bat) inside.

4
Get your Anthropic API key
One time
▼

The app calls Claude (by Anthropic) to adapt your CV. You need your own API key. It is free to start.

  1. Go to console.anthropic.com
  2. Sign up with your email and verify your phone number
  3. In the left sidebar, click API Keys
  4. Click Create Key, name it anything (e.g. "cv-adapter")
  5. Copy the key — it starts with sk-ant-api03...
Save the key now. Anthropic only shows it once. If you lose it, just create a new one.

Cost: Each CV generation costs roughly $0.02-0.05. New accounts receive $5 in free credits — enough for 100+ CVs.

You will paste this key into the app's sidebar when it opens. You can also save it so you do not have to paste it every time — there is an optional step for that in Step 6.

5
Install dependencies
One time
▼

This installs all the Python packages the app needs. You only do this once.

Terminal — make sure you are inside the cv-adapt-engine folder 
cd ~/Desktop/cv-adapt-engine
chmod +x install.sh run.sh
./install.sh

You will see packages downloading. It takes 1-3 minutes. When you see "Installation complete." you are done.

Command Prompt 
cd %USERPROFILE%\Desktop\cv-adapt-engine
install.bat

Windows may show a security dialog — click Run anyway. Wait for "Installation complete."

If you see errors about python3 -m venv or pip, re-check that Python was installed correctly (Step 2) and try again.
6
Launch the app
Every time
▼

Every time you want to use CV Adapter, run this command:

Terminal 
cd ~/Desktop/cv-adapt-engine && ./run.sh
Command Prompt 
cd %USERPROFILE%\Desktop\cv-adapt-engine && run.bat

The terminal will print:

Output
Starting CV Adapter at http://localhost:8501

Your browser should open automatically. If not, open it manually and go to http://localhost:8501.

Optional: save your API key so you do not paste it every time.
Create a file named .env inside the cv-adapt-engine folder with this one line:

.env (create with any text editor — Notepad, TextEdit, VS Code) 
ANTHROPIC_API_KEY=sk-ant-api03-your-key-here

Replace the value with your actual key. The app loads it automatically on next launch.

Press Ctrl+C in the terminal to stop the app when you are done.

7
Generate your first adapted CV
Every time
▼
  1. Enter your API key in the sidebar field on the left of the app. The Generate button will activate.
  2. Upload your CV (PDF or plain text file) — or paste the text directly into the "Your CV" tab.
  3. Add a job description — upload or paste any job posting you want to target.
  4. Fill in your contact info: name and email are required. Phone, LinkedIn, city, and country are optional but appear in the CV header.
  5. Select the ATS platform if you know which system the company uses (Greenhouse, Ashby, Workday, etc.). If unsure, leave "Auto-detect".
  6. Click "Generate Adapted CV". Wait 15-30 seconds.
  7. Review the results: you will see an estimated ATS match score and a gap breakdown.
  8. Download the PDF and open it to review. It should be 1-2 clean pages.
The app has two modes depending on the role:
  • Technical roles (engineering, data, DevOps): rewrites bullets aggressively to match ATS keyword patterns.
  • Non-technical roles (operations, HR, finance, marketing): preserves your original language and tone, only adjusts specific keywords. This is intentional — AI-based ATS systems penalize over-optimized prose in non-technical roles.

Common questions:

How do I stop the app?
Press Ctrl+C in the terminal window that is running the app.
Is my data private?
Your CV and JD text are sent to Anthropic's API to generate the output. Nothing is stored by this app locally beyond the session. Anthropic's privacy policy applies.
How much does it cost?
About $0.02-0.05 per CV. New Anthropic accounts get $5 free credits, which is enough for 100+ CVs.
The PDF came out as 3 pages — how do I fix it?
Your CV text is very long. Try pasting a shorter version — remove very old roles or shorten bullet points. The app works best with CVs that are already roughly 2 pages of content.
How do I get updates when new versions are released?
Inside the cv-adapt-engine folder, run: git pull. That fetches the latest version.
