How It Works

Three layers. Zero friction for legitimate applicants. Zero automated rejections.

Layer 1

Behavioral Baseline

Every application submission carries a fingerprint: timing patterns, tool signatures, device signals. Sentriq reads these passively at submission with zero friction and no changes to the application form.

  • Automation and tool fingerprinting
  • Behavioral anomaly detection
  • Device and environment signals
  • Cross-application pattern matching
Layer 2

Async Challenge

Triggered only when Layer 1 flags an anomaly. A short async task detects proxy impersonation and AI-assisted responses: legitimate applicants proceed unimpeded, proxies fail.

  • Triggered only on anomalous applications
  • Tailored to the specific role and application
  • No recruiter involvement required
  • Zero friction for legitimate applicants
Layer 3

Human Review

Triggered only when Layer 2 flags divergence. A Sentriq analyst reviews the full signal stack and surfaces a trust signal alongside the application: zero automated rejections, full audit trail, employer decides.

  • Zero automated rejections
  • Full audit trail per application
  • Employer retains final decision
  • Annex III EU AI Act compliant
What it looks like in your hiring flow
Each application arrives with a trust signal. High-risk applications trigger a parallel alert before recruiter time is spent; normal applications proceed with zero delay.
