Three-Layer Architecture
Layer 1

Application Integrity (Passive)

Behavioral metadata captured as the application is filled. Typing dynamics, mouse movement, fill patterns, mass-apply fingerprints, CV-to-JD overfitting, AI content markers. Invisible to legitimate applicants. No friction added.

Easy Apply exception: When candidates submit via LinkedIn Easy Apply or API bypass, Layer 1 has no form surface to run on. Easy Apply automatically triggers Layer 2. This is architecture, not a gap.

Layer 2

Async Personalized Challenge

Triggered only when Layer 1 surfaces anomalies, or on all Easy Apply submissions. A short task framed as part of the application. Scoring combines behavioral signals and content quality, calibrated to the profile the applicant presented. A proxy cannot replicate both the behavioral fingerprint and the knowledge of the person they impersonate.

Layer 3

Human Review

Sentriq analyst reviews the full signal stack. Employer makes the final call. Sentriq never rejects an application unilaterally. Full audit trail for EU AI Act Annex III compliance (deadline Aug 2, 2026). This is a mandatory product component, not an option. EU AI Act requires human oversight for all hiring AI systems.
