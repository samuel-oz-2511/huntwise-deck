How Sentriq Works
Layer 1

Behavioral Baseline

Passive, invisible. Every application submission carries a fingerprint: timing patterns, tool signatures, device signals. Captured at submission with zero friction and no changes to the application form.

  • No candidate friction. No form changes.
  • Works with any ATS or custom intake
  • Detects proxy submission, AI-assisted fill, bulk tooling
Layer 2

Async Challenge

Triggered only when Layer 1 flags an anomaly. A personalized async task designed to detect proxy impersonation and AI-assisted responses. Legitimate applicants proceed unimpeded.

  • Fired only on anomalous applications
  • Questions are AI-proof by design
  • Zero automated rejections at this stage
Layer 3

Human Review

When Layer 2 flags divergence, a Sentriq analyst reviews the full signal stack. A trust signal is surfaced alongside the application. The employer decides. Full audit trail at every stage.

  • Zero automated rejections at any stage
  • Every flag includes a plain-language reason
  • EU AI Act compliant by design
